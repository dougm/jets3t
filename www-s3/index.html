About JetS3t

JetS3t is an open-source Java toolkit and application suite for the Amazon Simple Storage Service (Amazon S3).

The JetS3t toolkit is intended to provide Java programmers with a powerful yet simple API for interacting with S3 and managing data stored there.

There are also four applications included in the JetS3t suite:

  • Cockpit An easy-to-use graphical application for transferring files, viewing and managing the contents of an Amazon S3 account.
  • Synchronize A command-line application for synchronizing directories on your computer with an Amazon S3 account. Ideal for performing back-ups or synchronizing files between different computers.
  • Uploader A graphical application that S3 account holders may provide to clients or customers without S3 accounts. Uploader allows users to upload files to S3 using a simple wizard-based workflow, while all uploads are authorized by a Gatekeeper service
  • Gatekeeper A servlet that runs on an S3 account holder's server and processes requests from client applications such as Uploader. The Gatekeeper authorizes the client application to perform operations S3 accounts such as uploads, downloads and deletes.

These applications are built using the JetS3t toolkit but you do not need to be a Java programmer to use them.

Using JetS3t

You need an S3 account with Amazon to use JetS3t - this account gives you access to the cheap and unlimited storage space that makes S3 so attractive. You can read more about the service and prices at the Amazon S3 home page.

To create an account you will need to sign up with Amazon.

If you already have an S3 account, you can:

  • Download the whole JetS3t suite from the Downloads page, or
  • Use the convenient online versions of Cockpit and Uploader
