Provides Cockpit, a graphical application for viewing and managing the contents of an Amazon S3 account.

Cockpit offers the following capabilities:

  • Create and delete buckets in your account
  • Manage the contents of your own buckets, and publicly-accessibly third-party buckets
  • Upload files to S3 using drag-and-drop, download them using the menus
  • Change access permissions for S3 buckets and objects
  • Automatically gzip and/or encrypt files sent to S3
  • Automatically gunzip and/or decrypt files retrieved from S3
  • Store your AWS login credentials in encrypted files

The manager can be run as a stand-alone application or deployed as an applet.
