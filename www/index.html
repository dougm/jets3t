What is jets3t?

jets3t is a Java (1.4+) toolkit for the Amazon Simple Storage Service (also known as Amazon S3). Building on the Java library provided by Amazon, the toolkit aims to simplify interaction with S3 while providing powerful additional features.

If you have an S3 account, you can use this toolkit to build applications that transfer files to and from your online storage area.

Toolkit Features:

  • Works with JDK version 1.4
  • Free, Open Source library (see licensing)
  • Easy to use (see Getting Started)
  • Built-in multi-threaded service engine that allows multiple uploads/downloads to run simultaneously
  • Includes utility classes to help encrypt and/or compress data
  • Includes a powerful REST/HTTP implementation based on HttpClient 3, and a basic SOAP implementation based on Axis 1.4.
  • Extensible Service interface, allowing developers to add their own REST or SOAP implementations based on their preferred libraries
  • Two example applications built using the toolkit

Included Applications

The jets3t project includes two Java applications built using the toolkit:

  • Cockpit: A graphical (Swing) application for viewing and managing the contents of an Amazon S3 account.
  • Synchronize: A console application to synchronize directories on your computer with an Amazon S3 account.

Alternate resources

There are a number of other programs and toolkits available for the Amazon S3 service. If jets3t doesn't meet your needs, perhaps something more appropriate will be available in Amazon's S3 Solutions Catalog.

Who should use jets3t?

The jets3t toolkit is intended to be used by Java developers, however the applications included with the project may also be of use to non-developers.

Project Goals

jets3t was started as a hobbyist project so we could build simple applications to backup important files to S3 (Synchronize), and easily manage the contents of our S3 account (Cockpit). We are releasing these applications, and the underlying toolkit, in the hope it may be useful for others.

We have no particular goals for the jets3t project at present except for fixing bugs and making minor improvements. If the project proves to be useful to others and there is sufficient demand, we will look into making larger scale changes/additions based on your suggestions.

So if you use jets3t and would like it to be improved to better meet your needs, please let us know.

What is Amazon S3?

The Amazon Simple Storage Service is a paid-for online file storage service offered by Amazon. The chief benefits of S3 are:
  • It's quite cheap
  • You can store unlimited files, including large files (up to 5GB)
  • Amazon handles the underlying distributed storage network, so you don't have to

For more details, see the Amazon S3 home page.

If you're ready to give it a try, you can sign up right now.

Why is this toolkit called jets3t?

jets3t stands for Java S3 Toolkit. jets3t is the closest thing to real word containing these "letters" we could think of.
