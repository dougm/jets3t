What is jets3t?

jets3t is a Java toolkit for the Amazon Simple Storage Service (also known as Amazon S3). If you have an S3 account, you can use this toolkit to transfer files to and from your online storage area and do other nifty things.

The jets3t toolkit is mainly aimed at developers, but the project also includes a couple of applications built on the toolkit that may be of use to everyone else:

Cockpit: A graphical (Swing) application for viewing and managing the contents of an Amazon S3 account.

Synchronize: A console application to synchronize directories on your computer an Amazon S3 account.

What is Amazon S3?

The Amazon Simple Storage Service is a paid-for online file storage service offered by Amazon. The chief benefits of S3 are:
  • It's quite cheap
  • You can store unlimited files, including large files (up to 5GB)
  • Amazon handles the underlying distributed storage network, so you don't have to

For more details, see the Amazon S3 home page.
